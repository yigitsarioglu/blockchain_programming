MyGov Token

An autonomous decentralized governance token contract

Governance token allows crowds to collectively manage a decentralized autonomous organization (DAO).

Anyone who owns at least 1 MyGov token is MyGov member

Submittting Survey costs 2 MyGov tokens and 5 USD stable coin.

Submitting Project Proposal costs 5 MyGov tokens and 50 USD stable coin

Donations can be accepted in USD stable coin and MyGov tokens only. USD tokens can be granted to winning Project proposals. MyGov tokens are given away via faucet function.

MyGov tokens are distributed via a faucet function. Faucet gives 1 token to an address. If the address has obtained a token before, it cannot get token from faucet any longer
